About

Dibangun untuk Menghentikan
Pencurian Data dari Dalam

Fortrix adalah Endpoint Protection Platform buatan Indonesia yang memantau aktivitas mencurigakan di setiap komputer dalam jaringan Anda. Berbeda dari antivirus tradisional yang fokus pada malware dari luar, Fortrix mendeteksi pola data exfiltration — seseorang atau program yang mencoba mencuri data dari dalam.

🌍

Teknologi Kelas Dunia

Engine Fortrix digunakan oleh perusahaan cyber security terkemuka di Eropa dan Amerika Serikat. Kini tersedia untuk pasar Indonesia.

Cara Kerja

Bagaimana Fortrix Melindungi Anda

1

Install Agent Ringan

Satu file EXE (6MB), tidak ada dependency. Install dalam 1 menit di setiap endpoint Windows. Agent berjalan sebagai service, tidak mengganggu pengguna.

2

Pantau Perilaku

Agent memantau 4 sinyal utama: proses baru, koneksi jaringan keluar, volume pembacaan file, dan frekuensi clipboard. Setiap detik, bukan setiap menit.

3

Deteksi & Alert

Rules engine menganalisis setiap event. Jika terdeteksi pola mencurigakan — seperti proses membaca 500MB file dalam 1 menit — alert otomatis dibuat dan email notifikasi dikirim ke tim keamanan Anda.

4

Investigasi & Tindakan

Dashboard web memberikan visibilitas penuh: device mana, proses apa, jam berapa. Tim Anda bisa langsung investigasi dan mengambil tindakan sebelum data keluar.

FAQ

Pertanyaan Umum

Apakah agent memberatkan komputer? +

Tidak. Agent Fortrix sangat ringan (6MB, <1% CPU, ~20MB RAM). Berjalan sebagai background service tanpa mengganggu pekerjaan pengguna.

Apakah Fortrix membaca isi file saya? +

Tidak pernah. Fortrix hanya mengukur volume data yang dibaca (berapa MB), bukan isinya. Kami tidak mengumpulkan isi file, keystroke, screenshot, atau isi clipboard.

Apakah saya perlu server sendiri? +

Tidak harus. Fortrix menyediakan cloud dashboard yang bisa diakses melalui browser. Untuk enterprise, tersedia opsi on-premise — server di data center Anda sendiri.

Bagaimana cara mencoba? +

Daftar melalui form di bawah. Tim kami akan memberikan akses trial 14 hari — tanpa biaya, tanpa kartu kredit. Anda bisa langsung install agent dan melihat dashboard.

Apakah mendukung macOS dan Linux? +

Saat ini agent tersedia untuk Windows (10/11/Server). macOS dan Linux sedang dalam pengembangan — target rilis Q3 2026.
